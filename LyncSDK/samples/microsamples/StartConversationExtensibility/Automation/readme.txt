Automation Sample

Uses the Automation class in the Lync Model API to start a new conversation with one or more Lync contacts.

Sample location:
64Bit Operating System: %PROGRAMFILES(X86)%\Microsoft Lync\SDK\Samples\Automation
32Bit Operating System: %PROGRAMFILES%\Microsoft Lync\SDK\Samples\Automation

Features
- Uses the Automation API in Lync to start a new conversation with a set of Lync contacts.
- The conversation can be started with one or more modalities at the same time.
- Shows how to set up specific settings for each conversation modality.

Warnings:
- The sample runs in Visual Studio 2008 but the project file included in it is for Visual Studio 2010.

Prerequisites (for compiling and running in Visual Studio)
- .Net Framework 4.0 or above.
- Visual Studio 2008 or 2010
- Microsoft Lync 2013 SDK

Prerequisites (for running installed sample on client machines)
- Microsoft Lync 2013 must be installed and running.

Running the sample
------------------
1. Open Automation.csproj
2. Hit F5 in Visual Studio

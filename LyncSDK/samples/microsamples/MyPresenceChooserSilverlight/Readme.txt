MyPresenceChooserSilverlight Sample

This sample shows how to create a Lync My Presense Chooser Control and retrieve the presence

Sample location:
64Bit Operating System: %PROGRAMFILES(X86)%\Microsoft Lync\SDK\Samples\MyPresenceChooserSilverlight
32Bit Operating System: %PROGRAMFILES%\Microsoft Lync\SDK\Samples\MyPresenceChooserSilverlight

Warnings:
- The sample runs in Visual Studio 2010.

Prerequisites (for compiling and running in Visual Studio)
- .Net Framework 4.0 or above.
- Silverlight SDK v. 5
- Silverlight Tools for Visual Studio
- Visual Studio 2010 or above.
- Microsoft Lync 2013 SDK

Prerequisites (for running installed sample on client machines)
- Microsoft Lync 2013 must be installed and running.

Running the sample
------------------
1. Open MyPresenceChooserSilverlight.csproj
2. Hit F5 in Visual Studio

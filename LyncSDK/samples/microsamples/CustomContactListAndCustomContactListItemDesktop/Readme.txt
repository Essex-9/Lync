CustomContactListAndCustomContactListItemDesktop Sample

This sample shows how to create a Custom Contact List Item, assign a sip uri, and add to a Custom Contact List.

This sample uses the Lync Model API.

Sample location:
64Bit Operating System: %PROGRAMFILES(X86)%\Microsoft Lync\SDK\Samples\CustomContactListAndCustomContactListItemDesktop
32Bit Operating System: %PROGRAMFILES%\Microsoft Lync\SDK\Samples\CustomContactListAndCustomContactListItemDesktop

Warnings:
- The sample runs in Visual Studio 2010.

Prerequisites (for compiling and running in Visual Studio)
- .Net Framework 4.0 or above.
- Visual Studio 2010 or above.
- Microsoft Lync 2013 SDK

Prerequisites (for running installed sample on client machines)
- Microsoft Lync 2013 must be installed and running.

Running the sample
------------------
1. Open CustomContactListAndCustomContactListItemDesktop.csproj
2. Hit F5 in Visual Studio

Add/Remove Contacts Sample

This sample uses the Lync Model API to add or remove contacts to the "Other Contacts" group.

Sample location:
64Bit Operating System: %PROGRAMFILES(X86)%\Microsoft Lync\SDK\Samples\AddRemoveContacts
32Bit Operating System: %PROGRAMFILES%\Microsoft Lync\SDK\Samples\AddRemoveContacts

Features
- Add or remove a custom group in the Lync contact list.
- Handle Lync events to respond to changes in the contacts added or removed.

Warnings:
- The sample runs in Visual Studio 2010.

Prerequisites (for compiling and running in Visual Studio)
- .Net Framework 4.0 or above.
- Visual Studio 2010 or above.
- Microsoft Lync 2013 SDK

Prerequisites (for running installed sample on client machines)
- Microsoft Lync 2013 must be installed and running.

Running the sample
------------------
1. Open AddRemoveContacts.csproj
2. Hit F5 in Visual Studio

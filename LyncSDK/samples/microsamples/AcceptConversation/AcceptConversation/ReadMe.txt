Accept Conversation Sample

This application demonstrates the ability to Ignore/Accept IM Conversation.

Sample location:
64Bit Operating System: %PROGRAMFILES(X86)%\Microsoft Lync\SDK\Samples\Micro\AcceptConversation
32Bit Operating System: %PROGRAMFILES%\Microsoft Lync\SDK\Samples\Micro\AcceptConversation

Features
- Ignore or Accept P2P IM conversation

Warnings:
- The sample runs in Visual Studio 2008 but the project file included in it is for Visual Studio 2010.
- Copy the AcceptConversation folder to a user folder, outside of Program Files.
- Both Lync and the AcceptConversation project must run with the same priviliges.

Prerequisites (for compiling and running in Visual Studio)
- .Net Framework 3.5 or above.
- Visual Studio 2008 or 2010
- Microsoft Lync 2013 SDK

Prerequisites (for running installed sample on client machines)
- Microsoft Lync 2013 must be installed and running.

Running the sample
------------------
1. Open AcceptConversation.csproj file.
2. Hit F5
3. Send the message to this client from any other machine

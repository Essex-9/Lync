ButtonsSilverlight Sample

This sample shows how to create a user control with Lync buttons, assign a sip uri the button source, and add the user control to a listbox

Sample location:
64Bit Operating System: %PROGRAMFILES(X86)%\Microsoft Lync\SDK\Samples\ButtonsSilverlight
32Bit Operating System: %PROGRAMFILES%\Microsoft Lync\SDK\Samples\ButtonsSilverlight


Warnings:
- The sample runs in Visual Studio 2010.

Prerequisites (for compiling and running in Visual Studio)
- .Net Framework 4.0 or above.
- Silverlight SDK v. 5
- Silverlight Tools for Visual Studio
- Visual Studio 2010 or above.
- Microsoft Lync 2013 SDK

Prerequisites (for running installed sample on client machines)
- Microsoft Lync 2013 must be installed and running.

Running the sample
------------------
1. Open ButtonsSilverlight.csproj
2. Hit F5 in Visual Studio

JoinMeetingFromLobby Sample

This sample shows how to join a meeting from the lobby

Sample location:
64Bit Operating System: %PROGRAMFILES(X86)%\Microsoft Lync\SDK\Samples\JoinMeetingFromLobby
32Bit Operating System: %PROGRAMFILES%\Microsoft Lync\SDK\Samples\JoinMeetingFromLobby

Features
- Start a JoinMeeting conference using Lync Automation class

Warnings:
- The sample runs in Visual Studio 2010.

Prerequisites (for compiling and running in Visual Studio)
- .Net Framework 4.0 or above.
- Visual Studio 2010 or above.
- Microsoft Lync 2013 SDK

Prerequisites (for running installed sample on client machines)
- Microsoft Lync 2013 must be installed and running.

Running the sample
------------------
1. Open JoinMeetingFromLobby.csproj
2. Hit F5 in Visual Studio
